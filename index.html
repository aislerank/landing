No black box

How we measure — plainly

🛡
We query AI platforms via their official APIs with retrieval enabled, using a fixed set of 40–60 buyer-intent prompts per category, re-run weekly. Every report states which platforms, which query set, and the run date. API answers can differ from what one consumer sees in an app — we say so on every report rather than pretend otherwise. We never scrape consumer apps. Because every brand in a category is scored against the identical run, every comparison is apples to apples.
FAQ

Questions founders ask

Can you really influence what AI assistants recommend?

You can't bribe a model — but recommendations are driven by what models can see and verify: structured data, product feeds, comparison content, reviews, merchant-program participation. Brands that fix those get cited. That's the work.

How fast do results show up?

AI answers reshuffle within days of model and index updates — that's why we track weekly. Fixes typically start showing in 2–6 weeks, and we show you the movement instead of asking you to trust us.

Is this just SEO with new branding?

It overlaps — good structured data helps both — but the ranking systems, query patterns, and fix lists are different. Your SEO agency isn't checking whether Perplexity recommends you for “best magnesium glycinate.”

Why supplements only?

Depth beats breadth: one category-wide query set means you're scored against your actual competitors, not generic benchmarks. Pet brands are next — join the list and we'll tell you when.
